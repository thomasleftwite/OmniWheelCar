# OmniWheelCar
Omuni Wheel Car by micro:bit and DFR0548
# Platform
Microsoft MakeCode for microbit
# Extension
https://github.com/DFRobot/pxt-motor
# Source Code Management
- original: *.ts files in this directory
- execution: paste those contents into platform on JavaScript mode
- flash the code via the platform to microbit
# Source Code
- controller.ts: control Omni Wheel Car via Gazell - microbit radio function.
- receiver.ts: receives controller commands via Gazell and drive omni wheels via DRF0548 - Microbit Driver Extension Board.
